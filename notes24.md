福音创造了截然不同的群体
- 因为主耶稣舍命的爱，所以我们有了服侍而非自私的关系。
- 它消除了恐惧和骄傲，所以那些根本无法相处的人在教会群体中也能和睦相处。
- 它呼召我们追求圣洁，所以神的子民生活在相互问责和管教的爱的纽带中。

- 恩典、智慧和品格的成长主要不是发生在课堂和教导中，不是通过大型敬拜聚会，甚至不是在独处中。
- 成长发生于深刻的关系中。
- 在社群中，福音的含义在认知上被阐明，在实践中被应用。
- 在恩典和圣洁中成长，成为门徒，最主要的方式是在基督徒群体当中，在深入参与到神的家中。

社群不是和传福音、外展、培训/门训、祷告、敬拜并列的事工，而是贯穿其他事工的方式。社群本身是我们进行外展和门训，甚至经历与神相交的主要方式之一。

## 社群的塑造力
通常的课堂关系虽然可以传递知识，但师生之间仅局限于知识层面的关系，缺乏同吃、同住等情感上深度的联结。相比之下，耶稣与门徒一起吃、住，同工同行，在讨论、对话和应用中产生深度的影响。成长发生于深刻的关系中，在社群中。

我们主要的社交群体对我们品格的形成有决定性的影响。
> - 讨论（dialogue）- 双向交流，而非单向灌输
> - 对话（discussion）- 深度探讨
> - 应用（application）- 在生活中实践

## 如何平衡"复兴主义"和"教会传统"

这一段提出了一个新概念：**"教会性复兴主义"（Ecclesial Revivalism）**

### 历史背景和问题诊断

#### **250年来的教会史分裂：**

**两种对立的"敬虔"（piety）模式：**

| 教会性敬虔 (Churchly Piety) | 个人主义敬虔 (Individualistic Piety) |
|---------------------------|----------------------------------|
| **强调：** | **强调：** |
| • 洗礼、圣餐等圣礼 | • 个人灵修和属灵操练 |
| • 顺服牧者和长老 | • 小组团契（很少长老监督） |
| • 教理问答和信仰告白 | • 个人见证和服事 |
| • 加入会员、公开宣信 | • 参与泛福音派的各种事工 |
| • 集体崇拜 | • 个人与神的关系 |
| • 通过教会的机构宣教 | • 个人布道和见证 |
| **危险倾向：** | **危险倾向：** |
| 形式主义、律法主义 | 主观主义、反智主义 |
| "只要受洗就得救" | "谁需要教会？我自己和神就行" |

#### **这种分裂是如何产生的？**

##### **复兴主义的贡献（18世纪开始）：**

✅ **正确的洞见：**
- 可能有人受了洗、加入了教会，但从未真正悔改归信
- 他们依赖"教会成员身份"而非真正信靠基督
- 需要呼召人自我省察、真实归信

❌ **过度反应导致的问题：**
- 削弱了教会制度的必要性
- 过分强调个人直接经验和自我判断
- 轻视圣礼（洗礼、圣餐）的价值
- 导致神学简化、情感主义、反智倾向

**历史学家John Coffey总结复兴主义的副作用：**
1. 用神学极简主义代替强健的信仰告白
2. 强调心灵体验，轻视正式的教会制度
3. 用危机性决志代替圣礼的日常滋养(归信不应只是某次聚会的决定，而应该是在教会生活、圣礼滋养、群体关系中的渐进成长)
4. 用民粹化的简单讲道代替学术性的神学讲道
5. 用天真的"圣经主义"代替历史神学的智慧

### 普林斯顿神学家的平衡方案

#### **Archibald Alexander和Charles Hodge的智慧：**

他们站在**中间地带**：

**一方面：**
- ✅ 接受复兴主义的基本洞见（需要真实归信）
- ✅ 跟随爱德华兹，教导如何分辨真假属灵经历

**另一方面：**
- ✅ 把教会放在基督徒生命的中心
- ✅ 强调圣礼、会员制、牧者权柄的重要性
- ✅ 批判过度的复兴主义（如Charles Finney）
- ✅ 也批判过度的圣礼主义（如John Williamson Nevin）

### "教会性复兴主义"的六个实践

这一段的**核心部分**：如何在今天的教会中实践这种平衡？

**1. 传讲归信，但尊重教会成员身份**
- 讲道中呼召人省察自己是否真正归信
- 但不随意宣称某个教会成员"未得救"
- 只有通过正式的教会纪律才能做此判断

**2. 审查入会候选人**
- 不要求人说出具体归信时刻
- 不要求所有人有相同模式的经历
- **要看：** 
  - 对罪和恩典有"属灵光照"式的理解吗？
  - 生命有"全面影响"（整体改变）吗？
  - 而不只是教义认同 + 道德行为

**3. 恢复教理问答**
- 长期的（几个月）、群体性的、参与式的门训
- 使用历史性总结（使徒信经、十诫、主祷文）
- 与公开崇拜结合，在主日认可和祷告

**4. 认识到慕道者需要过程**
- 在后基督教社会，人们需要长时间理解福音
- 需要看见信仰的活出，而非只听信息
- 崇拜本身应该既能传福音又能造就信徒

**5. 让洗礼和接纳会员更有教导性**
- 不要匆忙完成仪式
- 要求完成教理问答课程
- 公开见证、会众参与、神学教导
- 成为难忘的属灵里程碑

**6. 利用圣餐前的预备季**
- 在圣餐前1-2周聚焦某个实践领域
- 呼召群体性的自我省察和更新
- 圣餐成为这个过程的高潮
