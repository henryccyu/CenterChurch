# 教会植堂作为运动动力 － 要点

## 核心原则

**自然的植堂心态**
- 植堂应该是持续、自然的事工，而非创伤性的偶发事件
- 保罗的事工包含三个阶段：传福音、融入群体、培养领导力
- 植堂应该像敬拜、传福音和团契一样，成为常规事工的一部分

**三个关键的思维转变**
1. **愿意放弃资源和控制权** - 交出金钱、成员和领袖
2. **愿意放弃对事工形式的控制** - 新教会不会是母会的复制品
3. **更关心神的国度而非自己的团体** - 庆祝整体增长而非哀叹局部损失

## 为什么需要植堂

**植堂的多重益处：**
- **最有效的传福音方式** - 新教会从非信徒中吸纳的成员比例是老教会的6-8倍
- **更新现有教会** - 带来新思想、培养新领袖、激发自我反省
- **触及多样性** - 最好地接触新世代、新居民和新族群
- **自我维持的事工** - 几年内成为资源提供者而非接受者

**"1%规则"**
- 教会联盟每年应植堂数量达到现有教会总数的1%才能避免衰退
- 要在一代人内增长50%以上，需要每年植堂2-3%

## 实际影响

文档通过城市对比说明：
- **城市A**（所有教会超过20年）- 基督徒人数萎缩
- **城市B**（10%教会不到10年）- 基督徒人数持平或缓慢下降
- **城市C**（25%教会不到10年）- 一代人内增长50%

## 植堂四阶段

1. **学习（Learn）** - 了解社区、文化和世界观
2. **爱（Love）** - 培养健康的灵性和门徒关系
3. **连接（Link）** - 制定处境化策略
4. **启动（Launch）** - 实施具体行动计划

# 自然植堂详解

## 一、自然植堂的圣经基础

### 使徒行传中的模式

文档指出，在使徒行传中，植堂**不是创伤性或不自然的事件**，而是融入事工的经纬之中，稳定而正常地发生。

**保罗的三阶段事工模式**（使徒行传14章）：

1. **传福音阶段**
   - 使用希腊词 "euangelizo"（传福音）
   - 不仅是讲道，而是全方位的福音传播
   - 包括：会堂讲道、小组查经、市场演讲、租用大厅讨论、一对一交谈

2. **融入群体阶段**
   - 立即去坚固和鼓励归信者（徒14:22）
   - 教导"信仰"（明确的信念和神学体系）
   - 将新信徒"会众化"——带入定期聚会的群体

3. **领导力发展阶段**
   - 在每个地方选立长老
   - 从归信者中选出复数领袖
   - 给予他们教导和牧养的责任
   - 离开时，门徒变成了"教会"（徒14:23）

### 关键观察

**"在使徒行传中，教会的倍增就像个人归信的倍增一样自然"**

保罗从不单独传福音和门徒训练而不植堂。这应该像敬拜、传福音、团契、教育和服务一样，成为持续进行的自然事工部分。

## 二、非自然植堂的两种形式

文档警告了今天普遍存在的两种"非自然"植堂方式：

### 1. 对抗性植堂（Defiant Church Planting）
- **起因**：因教义、异象或事工理念的疏离而分裂
- **特征**：
  - 领袖个性冲突
  - 文化分裂（第二代领袖反对第一代）
  - 违背原教会领袖意愿

### 2. 勉强植堂（Reluctant Church Planting）
- **起因**：环境迫使领袖违背意愿植堂
- **例子**：
  - 教会建筑不够用
  - 成员搬到新区域要求建立类似教会
  - 不同异象的成员（年轻人、不同敬拜风格）开始流失
- **问题**：虽然可能给予支持，但**不太可能再次发生**——除非环境再次迫使

**这些是"非自然"的，因为植堂不会成为常规模式**

## 三、植堂的两种基本途径

### 1. 开拓性植堂（Pioneer Church Planting）

| 特征 | 描述 |
|------|------|
| 领袖 | 通常是自发起者 |
| 核心成员 | 无；通过社交网络和传福音获得 |
| 资金来源 | 宣教机构/个人筹款/带职/以上组合 |
| 导师 | 远程牧者，见面少；或只读书学习 |
| 模式 | 创新，开创新模式或模仿远程模式 |

**例子**：保罗和同伴——虽然安提阿教会差派他们，但在每个城市的工作本质上都是开拓性的。

### 2. 教会植堂教会（Church-Led Church Planting）

| 特征 | 描述 |
|------|------|
| 领袖 | 由教会选立，也可呼召差派 |
| 核心成员 | 来自(1)集合小组 (2)分出远距离家庭 |
| 资金来源 | 核心组认捐/母会资助/外部捐赠/组合 |
| 导师 | 与附近导师定期见面；常有同侪督导 |
| 模式 | 类似母会，但不完全相同 |

**新约例子**：初期教会的家庭教会模式
- 吕底亚归信后，她的家成为腓立比第一个教会（徒16:40）
- 基利司布的家庭（徒18章）
- 教会只能通过倍增新的聚会或家庭教会自然增长

## 四、自然植堂心态的三个转变

### 转变1：愿意放弃资源和失去控制

**必须放弃的：**
- 金钱
- 成员
- 领袖
- 人数
- 荣耀
- 直接控制权

**保罗的榜样**：
- 他"赋权"给新领袖
- 给予他们主权
- 交出大量控制权

**实际案例**：
某福音派教会占据历史小建筑，连续四年满座100人，抵制植堂怕失去人和钱。最终他们派50人到新镇建立新教会：
- **两年后**：子会达到350人
- **母会**：三周内再次满座
- **反思**：本可植三个教会，共近千人的教会家庭

**挑战**：
- 如果你的事工吸收人进入你的教会，会增加你的人数、控制和荣耀
- 如果你组织新人进入新教会，你会失去金钱、成员、人数、领袖和控制
- **但这正是保罗所做的！**

**额外问题**：
- 放手后我们失去直接控制，但不能避免问题的责任
- "有点像成年子女的父母——不能直接告诉他们该做什么，但出问题时要帮忙收拾"

### 转变2：愿意放弃对事工形式的控制

**现实**：
- 新教会**不会看起来像原来的教会**
- 会发展自己的声音和重点

**平衡的智慧**：

**一方面** - 必须确保差异不太大
- 使徒行传讲到"信仰"（the faith）
- 基督教核心有一套真理教义
- 否则团契和合作会紧张

**另一方面** - 不能坚持新教会必须是克隆
- 这表明你不愿承认处境化的现实
- 不同世代和文化会产生不同类型的教会
- 这不会削弱母会的健全性，反而证明它

**关键洞察**：
> "自然植堂心态不是信任新领袖的问题，而是**信任神**的问题"

- 保罗将他们"交托给主"（徒14:23）
- 保罗的心肠和性格使他不需要保持控制
- 他相信神会继续他在教会中开始的工作
- **需要高水平的属灵成熟和对神护理的信靠**

### 转变3：愿意更关心神的国度而非自己的部落

**保罗的榜样**：

1. **对待亚波罗**（徒18:24-28）
   - 以最温暖的措辞称呼他（林前3:6; 4:6; 16:12）
   - 尽管亚波罗不是他的门徒
   - 尽管亚波罗的门徒视自己为独特派别（林前1:12; 3:4）

2. **愿意放手**
   - "然后他们离开了"（徒16:40）
   - 不关心自己或自己派别的权力
   - 关心整个国度

**现实测试**：

当社区有新教会时：
- 新教会大部分新成员来自非信徒
- 但也会吸引一些现有教会的人

**选择时刻**：
当我们失去2-3个家庭，而新教会带来100个原本不去任何教会的人时...

**我们要问自己**：
> "我们是要庆祝国度通过这新教会得到的新人，还是要哀叹和怨恨我们失去的家庭？"

**态度揭示**：
- 哀叹小损失而非欢庆大收获 = 狭隘利益
- 对新教会发展的态度 = 测试我们的心态是关注自己的地盘还是城市中神国度的整体健康和繁荣

## 五、为什么植堂必须是自然的

### 1. 它是第五个"事工前线"

文档提出植堂与整合事工的四个方面并行：
- 连接人与神（敬拜和传福音）
- 连接人与彼此（门徒训练和群体）
- 连接到城市需要（公义和怜悯）
- 连接到文化（整合信仰和工作）
- **第五个**：将教会倍增成拥有其他四个前线的新教会

### 2. 初期教会的结构使其自然

**家庭教会是基本建构单元**
- 教会只能通过倍增新聚会或家庭教会自然增长
- 虽然保罗写信给"哥林多教会"（单数）
- 但显然是在写给多个家庭教会（革来氏、司提反等）
- **植堂内置于教会本质中**

### 3. 符合使徒行传的模式

**文档指出一个隐含的"终止论"**：
- 读使徒行传的人无意识地说："是的，但那是当时。我们现在不需要那样做。"
- 但植堂在使徒行传中与其他所有事工一起出现
- 为什么列出事工基本要素时总是忽略植堂？

### 4. 当代世界类似使徒行传时代

**我们的世界再次像希腊罗马世界**：
- 多元、充满活力的宗教信仰社区和选择
- 传统、世俗和异教世界观并存
- 城市是有影响力的文化中心
- 城市变得多民族化和全球连接
- 我们生活在"使徒行传式"的世界而非基督教世界

**因此**：
> "植堂必然像对前几代人一样，成为接触我们世界的核心策略"

## 六、终极榜样：耶稣

**耶稣是终极的植堂者**：
- 他建造他的教会（太16:18）
- 他有效地做，因为地狱之门不能胜过
- 他兴起领袖，给他们天国的钥匙（太16:19）
- 他将归信者建立在认信的使徒彼得的话语上——即神的话语上（太16:18）

**我们的角色**：
- 当我们植堂，我们参与神的工作
- 如果有任何成功，是因为"神使它生长"
- "栽种的算不得什么，浇灌的也算不得什么，只在那叫它生长的神"（林前3:6-7）

## 总结

**自然植堂的本质**：

1. **持续性**而非偶发性
2. **整合性**而非附加性  
3. **有机性**而非机械性
4. **国度导向**而非机构导向
5. **信靠神**而非控制

**核心信念**：
正如个人归信的倍增是自然的，教会的倍增也应该是自然的。这不是可选项或特殊时期的策略，而是健康、有机体化教会的本质表现。
