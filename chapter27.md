# 第27章 运动与机构

十九世纪的宣教事业为有效运动的特征提供了重要的见解。在那个世纪初，西方传教士在非西方世界建立的许多新教会陷入了不健康的依赖模式。这些会众和宗派拥有真教会的传统标志——忠实地宣讲神的话语、正确施行圣礼，以及一个运作中的纪律体系。¹ 他们坚持纯正的教义，并吸纳了当地的牧师和领袖，但他们却无法轻易地自我繁殖或在经济上自给自足。结果，他们无限期地依赖西方传教士和资金。

约翰·内维斯 (John Nevius)、戴德生 (Hudson Taylor)、罗兰·艾伦 (Roland Allen) 等人开创了一种替代性的宣教方法，旨在建立从一开始就自给自足的教会。目标是建立能够自然成长的会众，而不是依赖外国援助这种人为的“生命支持”。这不仅体现在他们有效地在自己的文化中赢得归信者，而且还以一种能够定期自我复制的速度吸引和培养新的本土领袖。简而言之，他们希望教会拥有一种内在的活力，使其能够从内部成长，而不需要外部的资金和领袖来支撑。他们希望这些教会不仅仅是健全的机构，更是充满活力和动力的运动。

罗兰·艾伦的书名《教会的自发性扩张》(The Spontaneous Expansion of the Church) 抓住了这个想法。² 它唤起了自燃的意象——无需外部点火源的燃烧。一个具有运动活力的教会（或一群教会）从内部产生自己的归信者、想法、领袖和资源，以实现其成为城市和文化教会的愿景。除非环境极其恶劣（例如，严重的迫害、战争、经济崩溃），否则教会在人数上和属灵成熟度上都会成长。用宣教学者的语言来说，这样的教会是“自传、自治和自养”的。它将复制出其他出于同样原因而自我复制的教会。汇集和部署的理念、领袖和资源越多，运动的动力就越强，并像滚雪球一样增长。只要复制的教会保持统一的异象，运动就能积聚动力并稳步（甚至指数级地）增长。

没有运动活力的教会就像一个靠生命支持机器维持生命的人。我见过至少三种教会**没有运动活力**却得以生存的方式：

1. **一些教会有宗派结构或宣教机构在经济上进行补贴。**
2. **一些教会拥有大量的捐赠基金和一个作为当地社区中心的建筑**。在这种情况下，短期内不需要外部的经济或领导援助，但教会不会产生额外的资源或活力来维持通过归信和成员属灵成长带来的增长。它本质上是作为一个运营良好的企业运作。资金来自审慎管理的捐赠基金，辅以租金、费用和少量捐款收入。许多教会以这种方式作为**机构**维持自己。
3. **一些教会在一个更大的、停滞不前的结构内，拥有一小群工作过度的人。**

> ### 上帝的主权与教会成长
> 
> 教会成长这个主题可能会引起争议。在引言中，我简要讨论了保罗将教会比作园子（林前 3:4-9）的比喻。在这个意象中，带领教会就像园艺。园子的成长取决于至少三个因素：（1）园丁的技巧和勤奋，（2）土壤的肥沃程度，以及（3）天气条件。如果熟练的事工像园艺，而土壤和天气条件类似于上帝圣灵的主权，那么我们看到，教会缺乏成长不能简单地归因于人的失败，也不能简单地被原谅为上帝的旨意。加尔文关于上帝主权的经典论述完全肯定了人类责任的重要性。³ 
>
> 泛泛而言，许多现代教会增长技巧的书籍没有充分重视上帝的主权，而许多反对教会增长的书籍和声音则为教会缺乏属灵活力找借口。我们在本章的观点是，一个教会可能是一个教义纯正的组织，但却不是一个充满活力、不断成长的有机体。《圣经》所描绘的教会是两者兼具的。 

尽管会众没有运动活力，但它却被内部支撑着。也就是说，一小部分人投入了过多的时间和资金来维持一个停滞或衰退的教会。这些人本身可能是属灵上充满活力的基督徒，但他们无法将这种活力传播给教会的其他人；或者他们可能只是勤奋工作、在教会有深厚根基的人，这产生了一种牺牲式的忠诚。这个解决方案是暂时的。在某个时刻，少数通过牺牲性奉献维持教会生存的人会因过于疲惫而无法继续，而由于缺乏自我复制的能力，教会最终会消亡。

## 运动与机构的对比

我并不是简单地暗示运动是好的，机构是坏的，而是说组织应该既具备机构特征，也具备运动活力，尽管在平衡上存在一些张力和取舍。机构通过变化缓慢的规则和政策来促进稳定的行为模式，从而限制和塑造人们的选择和做法。⁴ 但这种故意的选择限制往往是健康的。想一想杂货店。顾客通常很清楚如何结账。他们知道该去哪里、该站哪里；他们知道排队（队列）以及预计排队多长时间；他们也知道到达队伍前面时该做什么。如果你每周支付商品的方式都发生剧烈变化，那会是什么样子？那将是混乱的。制度化使得每天有数百万人能够在杂货店购买他们需要的东西。一些制度化的做法是正式的（比如如何付款），而另一些则是非正式的（比如人们预计等待多长时间）。如果你试图用一块金条在杂货店支付食物费用，那是行不通的。如果顾客不得不排队等待一小时，就会爆发愤怒。为什么？因为“每个人都知道”等待一小时太久了（至少在西方国家是这样）。杂货店有义务（非正式地）不让你等待那么久。如果它违反了这种信任，你可能就不会再来了。你的期望和行为已被这个机构限制、引导和塑造。如果没有制度化，没有人能高效地进入杂货店购物。

休·赫克洛 (Hugh Heclo) 这样定义机构：“机构代表着有价值目的的遗产，伴随着规则和道德义务，”由拥有权力的人来管理。⁵ 这是一个抽象的、学术性的描述，但它将我们引向一个有益的方向。机构依赖于对既定权威的服从，这种权威维护了过去的价值观和目的。机构是必要且有益的，它们提供了既定的、可靠的系统和框架来完成需要完成的事情。赫克洛写道：“生活在一个背弃机构的文化中，就相当于试图生活在一个没有骨骼的身体里，或者希望使用一种语言却不使用其语法。”机构为生活带来秩序，并为人类繁荣和文明社会建立了许多条件。

另一方面，运动更多地与**个人偏好的主张**和**带来未来的现实**有关。我将运动的四个关键特征定义为：**异象、牺牲、伴随统一的灵活性和自发性。**

**1. 首先，运动的标志是一个引人注目的异象**。异象包含对未来有吸引力、生动和清晰的图景，这是运动及其领袖寻求实现的。运动宣称：“如果这是你想去的地方，请与我们同行。”这种未来图景伴随着一套运动所承诺的强大价值观或信念。异象的内容必须表达出来，以便其他人能够轻易掌握；它不能过于深奥或困难，以至于只有少数人能够阐明它。

异象的内容必须表达得引人注目，以便其他人可以在不需要大量集中控制或协助的情况下，在自己的社区中学习并贯彻执行。例如，戒酒无名会“十二步小组”的变革性概念已在无数书籍中得到引人注目的表达和应用。因此，一个有异象通过这种小组改变生活的人，通常可以简单地拿起资料并开始。他们不需要任何人的许可或资金，而且有很多方法可以获得优秀的培训。或者，举一个不那么乐观的例子，我们可以注意到基地组织之所以有效，原因之一是它广泛而清晰地传播了其世界观。人们吸收它并用它来教育自己，许多人组建了在没有中央控制或通信的情况下运作的恐怖小组。在某些情况下，他们可能会去基地组织的训练营变得更有效，但之后，他们基本上被信任可以制定自己的地方战略。这些例子的重点是，戒酒无名会和基地组织是充满活力、不断发展的运动，而不是集中的机构。这是他们有效性的原因，也是他们能够以相对适度的资金实现增长的原因。异象成功的关键在于其**简单性**和**可用性**，通常以传递、阐释和应用异象的内容形式体现。

相比之下，尽管机构几乎总是有书面形式的目的声明（例如，学校是为了教育，企业是为了生产产品，医院是为了治愈病人），但将机构凝聚在一起的真正粘合剂是**它的规则、规定和程序**。在运动中，**共同的异象**指导着日常的选择；而在机构中，通常是**规则和既定模式**。

**2. 运动中统一的异象是如此引人注目，以至于它导致了一种牺牲性承诺和内在奖励的文化**。个人将异象置于自己的利益和舒适之上。在任何运动的早期，主要参与者通常无偿工作，不断生活在破产的威胁中。实现目标的满足是他们的主要回报。有些人将这称为“内在”奖励——来自知道自己是带来如此多益处的工具的**内在、个人成就感**。然而，在机构中，每个职位都有高度明确的权利和特权，以及清晰的薪酬和福利。机构的主要激励措施围绕着这些“外在”奖励。机构成员当然知道有工作要做，但他们的工作产出是根据具体的奖励仔细平衡的。**检验你的教会是否具有运动活力的最实际指标是检查你是否拥有牺牲的文化**。如果教会的最高领袖是唯一做出所有牺牲的人，那么你就没有运动文化。

**3. 运动的特点是对其自身成员之外的其他组织和人员采取慷慨灵活的态度**。运动将“**做什么**”（实现异象）视为比“**如何完成**”和“**由谁完成**”更高的价值。异象鼓励牺牲，运动的成员愿意结盟，与任何对异象感兴趣的人合作。另一方面，制度化组织更致力于**继承的实践、正确的程序和经过认证的人员**的重要性。他们经常选择不达成结果——尽管可能非常渴望——如果他们不能以规定的方式和由适当认证的当事方来完成的话。

我们在运动中发现的灵活性精神意味着**高度的统一**——在运动内部以及与其他组织的关系中。机构通常不鼓励这种类型的统一，即使在内部也是如此。它们往往由一套**重视自身利益的“各自为政”**的部门组成，每个部门都更关心自身的福祉，而不是整体的利益。通常，机构缺乏组织上的统一性，甚至可能对其他组织抱有敌意。

**4. 运动会自发地产生新的想法和领袖，并从内部成长**。机构就其本质而言，是为了长期持久性和稳定性而构建的，并且容易抵制有风险的新想法。但运动愿意承担新的风险，因为成员已经做出牺牲来成为工作的一部分。运动也倾向于吸引和奖励**能产生结果**的领袖。同样，原因在于实现异象非常重要。然而，机构由于重视稳定性和持久性，倾向于根据**任期**以及**被接受的资格和证书的积累**来奖励领导力。

总结运动和机构在其最强形式下的重要区别，有助于我们更清楚地看到其不同之处（参见下表）。

| 机构 (INSTITUTION) | 运动 (MOVEMENT) |
| :--- | :--- |
| 由**规则和程序**维系 | 由**共同目的、异象**维系 |
| 一种**权利和配额**的文化；责任与奖励的平衡 | 一种**牺牲性承诺**的文化 |
| 强调**薪酬、“外在”奖励** | 强调**庆典、“内在”奖励** |
| 政策变化涉及漫长过程、所有部门、大量阻力和谈判 | 异象来自**魅力型领袖**；被忠诚地接受 |
| 决策**程序化**且**缓慢** | 决策**关联化**且**迅速** |
| 创新**自上而下**；在部门壁垒中实施 | 创新**自下而上**；由整体执行 |
| 感觉像一个**各自为政的迷你机构或部门**的拼凑 | 感觉像一个**统一的整体** |
| 价值观：**安全、可预测性** | 价值观：**风险、偶然性** |
| **稳定、变化缓慢** | **充满活力、变化迅速** |
| 强调**传统、过去和习俗**；对未来趋势感到恐惧和否认 | 强调**现在和未来**；很少强调过去 |
| 工作分配给**有认证和任期**的人 | 工作分配给**产生最佳结果**的人 |

当我们看到这些对比鲜明的特征时，我们开始更好地理解为什么运动会**自发地产生**。运动能够产生新的想法，因为它鼓励人们集思广益，并且更愿意进行实验和尝试新想法。运动比机构“更扁平”——层级更少、壁垒更少——因此新想法能更快地获得推动力。运动也更善于培养新领袖，因为它们能吸引最有抱负和创造力的人。由于它们注重结果，因此可以迅速识别并提拔新兴领袖。运动成长得更快，因为它们对新想法的测试使其能够适应环境的变化。

## 运动与机构的融合

年轻的教会领袖可能会对运动感到兴奋，并大声疾呼反对体制化教会的盲目和僵化。事实上，任何浏览下表左栏的人都知道，太多的教会过于制度化了。哈佛学者大卫·赫斯特 (David Hurst) 很好地总结了运动如何变成机构——**异象变成战略、角色变成任务、团队变成结构、网络变成组织、认可变成薪酬。**⁷

然而，请记住，当我们看实际的例子时，在这两种形式之间划出过于尖锐的界限，甚至将它们如此鲜明地对立起来是错误的。尽管基督教运动文献有充分的理由对制度主义持高度批判态度，但留下的印象往往是所有权威、集中控制和正式程序都对事工不利。现实要复杂得多。首先，尽管新的教会和事工努力保持非正式、非编纂和非集中化，**但制度化是不可避免的。**

一旦我们做出一个选择——创造一项新政策、行政结构，或价值观和信念的共识——并开始将其带入未来，从而塑造人们的常规、期望和允许的偏好，我们就开始**将该价值观或信念制度化了**。

而且某些制度化甚至是可取的。正如前面所指出的，**统一的异象**——由运动的每个成员持有——对于运动活力至关重要。但这个异象不能每天或每年都改变，否则会在运动中造成混乱并阻碍其增长。具有讽刺意味的是，这意味着异象本身需要一定程度的**编纂和控制**。换句话说，维护运动活力的引擎——一个统一的异象——**需要采用机构的一些方面**。异象变成了运动所守护和传承的“传统”。

此外，我们注意到运动严重依赖其成员的牺牲性承诺，尤其是在刚开始时。在这种初创模式下，成员可能会刷爆信用卡并动用储蓄来启动事物。但这种生活方式是**不可持续的**。任何引人注目的异象都将是一个宏大的异象，而宏大的异象需要长期的努力——这种努力将需要，例如，带来足够的收入，以便创始人可以还清信用卡，并最终有足够的钱来生活和抚养家人。换句话说，**一个运动最终必须确立一个可持续的商业模式，该模式能产生足够的资源来支付开支。**如果它未能做到这一点，最终会耗尽最优秀的人，并无法朝着异象前进。

因此，一个强大、充满活力的运动占据了**中心**这个困难的空间——一个在**自由运行的有机体**和**有纪律的组织**之间保持张力和平衡的地方。一个拒绝承担某些组织特征——权威、传统、信念统一和质量控制——的运动将会**分裂和消散**。一个未能抵制向完全制度化不可避免趋势发展的运动，最终会**失去其活力和有效性**。运动领袖的工作就是驾驭这艘船，安全地航行在**这两种危险**之间。

**讨论和反思问题**

1. 如果你的组织突然必须搬离其建筑，被切断与宗派支持结构的联系，失去捐赠基金和银行存款，并经历了其资深领袖的离世，会发生什么？是否会留下一个有韧性的机构，能够靠着上帝的恩典重新振作起来，并从内部培养出新的领袖？如果不能，哪三种停滞结构（由外部补贴、由捐赠基金生命支持管理，或由一小群工作过度的人支撑）最能描述你的会众或组织？
2. 本章提出了运动的四个关键特征：异象、牺牲、伴随统一的灵活性和自发性。你在自己的事工或教会环境中是如何体验这些的？根据你的经验，这些特征中的每一个是如何促进运动活力的？
3. 回顾对比机构与运动的表格。当你考虑自己的教会时，你看到了哪些运动的特征？存在哪些机构的特征？你可以做些什么来鼓励你的教会增加运动活力？
